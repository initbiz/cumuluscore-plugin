# Upgrading from Cumuluscore v3 to v4

This guide can be used to help migrate form previous version of plugin to v4.

## Upgrade instructons

Cumuluscore v4 use RainLab User plugin v3. Backward compatibility with previous versions of User plugin is no longer maintained.
If you're not using User plugin in the newest version, in order to install Cumuluscore please update it, here's a upgrade guide:

https://github.com/rainlab/user-plugin/blob/master/UPGRADE.md
